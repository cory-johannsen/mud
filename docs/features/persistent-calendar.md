# Persistent Calendar

An in-game calendar that advances with time, driving time-of-day and seasonal events.

## Requirements

- [x] Persistent calendar
  - [x] Day & Month
  - [x] Displayed on screen
    - [x] Move time from prompt to room section, after the room name (on the same line as the room name)
    - [x] Add Month and day to time display in the format "January 1st", preceding the time of day
  - [x] Drives seasons and weather
    - [x] 4 seasons follow standard calendar rules
    - [x] Weather effects
      - [x] determined by season
      - [x] scale with room danger level [requires Room Danger Levels]
