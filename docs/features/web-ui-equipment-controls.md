# Web UI — Equipment Tab Controls

**Slug:** web-ui-equipment-controls
**Status:** backlog
**Priority:** 482
**Category:** ui
**Effort:** S

## Overview

The web UI Equipment tab should display equipped items per slot with an unequip control, and allow equipping items from inventory into empty slots, without requiring the player to type commands.

## Dependencies

- `web-client` — Equipment tab in the web game client
