```go
// ambientDoseInterval is the minimum time between ambient substance doses in a room (REQ-WK-50).
const ambientDoseInterval = 60 * time.Second

// ShouldApplyAmbientDose reports whether enough time has elapsed since lastDose
// to apply the next ambient substance dose.
//
// Precondition: now must be >= lastDose (or lastDose is zero).
// Postcondition: Returns true iff lastDose is zero OR now-lastDose >= 60s.
func ShouldApplyAmbientDose(lastDose, now time.Time) bool {
    if lastDose.IsZero() {
        return true
    }
    return now.Sub(lastDose) >= ambientDoseInterval
}

// tickAmbientSubstances checks each active player's current room for an
// AmbientSubstance. If found and 60s have elapsed since the last ambient
// dose, it applies one dose via ApplySubstanceByID (REQ-WK-50, REQ-WK-51).
//
// Precondition: called from the 5s ticker goroutine while holding no locks.
// Postcondition: Each eligible player receives at most one dose per 60s window.
func (s *GameServiceServer) tickAmbientSubstances() {
    now := time.Now()
    s.sessions.ForEachPlayer(func(sess *session.PlayerSession) {
        room := s.world.RoomByID(sess.RoomID)
        if room == nil || room.AmbientSubstance == "" {
            return
        }
        if !ShouldApplyAmbientDose(sess.LastAmbientDose, now) {
            return
        }
        sess.LastAmbientDose = now
        s.ApplySubstanceByID(sess.UID, room.AmbientSubstance)
    })
}
```

In `internal/game/session/manager.go`, add to `PlayerSession`:

```go
// LastAmbientDose records the last time an ambient substance dose was applied
// to this player. Zero value means never dosed (REQ-WK-50).
LastAmbientDose time.Time
```

In the 5s ticker goroutine (after the existing `roomRefreshTicker.C` case within `Session`), add a call to `tickAmbientSubstances` in the global zone ticker goroutine (`tickZone`). Specifically, in the `select` block for `roomRefreshTicker.C` in the per-player goroutine, add:

```go
s.tickAmbientSubstances()
```

> **Note:** `tickAmbientSubstances` is called once per 5s tick. Each call skips players that have not yet reached the 60s threshold, so the overhead is O(players) per tick regardless of zone count.
